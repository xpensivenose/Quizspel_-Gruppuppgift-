Quizspel

Detta är ett interaktivt quizspel som är byggt med JavaScript, HTML och CSS. Användaren får en fråga i taget, väljer ett svar och får direkt feedback om det är rätt eller fel. I slutet visas ett resultat baserat på antal rätt svar.

Funktioner

En fråga i taget
Klickbara svar
Visar rätt/fel direkt
Poängräkning
Visar slutresultat
Visar progression

Hur man kör Projektet

Ladda ner projektet som .zip
Öppna mappen i VS Code
Högerklicka på index.html
Välj Open with Live Server
Alternativ 2
Öppna index.html direkt i webbläsaren

Buggar

Efter man trycker på "Starta om quiz" visas inte poängen direkt vid start (visas efter första svar)
